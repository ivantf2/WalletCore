# WalletCore

##  Tema projekta
**Aplikacija za upravljanje osobnim financijama**

WalletCore je web aplikacija razvijena u Django frameworku koja omogućuje korisnicima praćenje prihoda i troškova, upravljanje budžetom te pregled financijske statistike kroz jednostavno i pregledno sučelje.
